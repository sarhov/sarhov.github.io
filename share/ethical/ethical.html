1. Settings for variants as separate product

For this go into Themes > Customize Theme > Generall Settings under Global Layout sections you can find the settings

3. Settings for images in inner pages

At first we will need metafields free app from chrome app store

Visiti the following link - https://chrome.google.com/webstore/detail/custom-fields-for-shopify/alfplfpobekffinigeidgmmfjollghln

After we install this we can activate additional field for inner pages and put there the value we need, in our case it will be url to the image which will be loaded as banner for each page.

Now we can upload our images into files section and then take their url and past in the fields we now have in dashboard, for this

we go into settings >  files and upload our images<

We now just need to copy the url of the image, go to the page we want this image be as banner, activate metafields and put the url into that field. Check please this videoinstruction where the all steps are shown.

3. Group images by their variant color

So let's provide some text insutrctions before the video, so how it is working, it works using ALT tag/field of the image and we can group them just defining their group/variant like this grouped_for_color for example it can be grouped_for_black - every images which is going to be grouped under color black should be has this tag.

Now check please the video where I am doing which I have described above.

4. Rollover the second variant image on hover

So, what was the problem with this rollovering, for the products where we don't have several variants the rollover is working properply, for the products with variants it did not, because it could find the second image of that variant, because we can have only one variant image.

To add this functionality I have used the same grouping method and when we hover over the product the script is searching the throught the all product image the image which has the same ALT text as the variant image.
